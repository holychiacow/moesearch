Privacy Policy for 萌萌查 Last Updated: March 2026 1. Data Collection This extension does not collect, store, or transmit any personal data, browsing history, or selected text to any external servers. 2. Data Usage Selected Text: When you highlight text, it is processed locally within your browser to facilitate the search. Local Storage: We use chrome.storage.local to remember your last searched word so the side panel stays in sync. This data never leaves your device. Third-Party Content: The extension embeds moedict.tw via an iframe to provide definitions. Your interaction with that content is subject to the privacy practices of MoeDict (g0v). 3. Permissions Host Permissions: Required only to detect text selection on the websites you visit. Storage: Used only to cache the current search term locally. 4. Contact If you have questions, please contact the developer via the GitHub repository or the Chrome Web Store support link.
